1. Download IntelliJ community version
2. Download JDK lastest version from oracle.com
3. Tutorial
   https://youtu.be/H_XxH66lm3U
   https://www.jetbrains.com/help/idea/creating-and-running-your-first-java-application.html#get-started

   IntelliJ -> Project Structure -> Artifaces -> + -> Jar
   pick "from modules with dependencies
   MenuBar -> Build -> Build artifact

5. Edit Configuration
   From IntelliJ MenuBar -> magnifying glass Search -> EditConfiguration
6. Press Control key twice to bring out the Run Anything dialog
