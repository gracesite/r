> docker run  --name=mysql -d mysql/mysql-server:latest   # -d : detached mode

Connecting docker mysql via port 


> docker pull mysql
> docker run --name=testsql -e MYSQL_ROOT_PASSWORD=xxxxxx -d mysql 
> docker inspect testsql | grep IPAddress
    # you should be able to connect to mysql with this ip address on port 3306
> docker pull phpmyadmin/phpmyadmin
> docker run --name myadmin -d --link testsql:db -p 8080:80 phpmyadmin/phpmyadmin 
open up a browser and go to ‘http://localhost:8080/’
